# Screenshots

Tento adresář obsahuje screenshoty aplikace pro README.

## Jak přidat screenshoty

1. Spusť aplikaci: `.\start-all.ps1`
2. Otevři v prohlížeči: `http://localhost:3000`
3. Udělej screenshoty:
   - **dashboard.png** - Dashboard stránka s přehledem
   - **telemetry.png** - Telemetry Analysis s porovnáním jezdců
   - **circuit.png** - Interaktivní okruh s rychlostmi
   - **laps.png** - Analýza časů na kolo

4. Ulož do tohoto adresáře s názvy uvedenými výše
5. Doporučená velikost: 1920x1080 nebo 1600x900

## Tipy pro kvalitní screenshoty

- Použij tmavý režim (lepší vzhled)
- Vyber zajímavý závod (např. Monza, Silverstone)
- Pro telemetry vyber kontrastní jezdce (např. Verstappen vs Hamilton)
- Ujisti se, že všechny grafy jsou načtené
- Screenshot celého okna prohlížeče (nebo jen obsah aplikace)

## Alternativa

Pokud nechceš dělat vlastní screenshoty, můžeš použít nástroje jako:
- Windows Snipping Tool (Win + Shift + S)
- ShareX (free screenshot tool)
- Nebo přímo v README použít GIF animace pomocí ScreenToGif
